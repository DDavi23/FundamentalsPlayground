Git Notes

Local

  • Locally means that things are happening just on my computer.
  • Local commit back up to a local repository and others can't access until we push to a remote.

Remote

  • Remote means your browser connects to a online server.
  • Remote can be accessed by fellow team memebers (private repo) or pubicly(public repo).

Local Set Up

  1. Open your command line/terminal.
  3. Use the cd command to mavigate to your folder. FOr example cd desktop/playground
  4. Type in git init. This will set up a repository(repo) for you locally
  6. A message like this should pop up: "Initialized empty git repository in dir/filename/stuff/here"
